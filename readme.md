### 1. requirements

-   nodejs LTS
-   angular cli (optional) `npm install -g @angular/cli`

### 2. local install

`npm install`

### 3. output bundle

`npm run build`

### or

### run in dev:

`npm run watch:chrome` in one terminal
`npm run watch:angular` in another one

### 4. select `angular/dist` directory on `load unpacked`

### 5. done.
